
# Vehicle Type Detection 🚗🚙🚌

**Vehicle Type Detection** is a personal AI project that uses deep learning and machine learning concepts to detect and classify vehicle types from images and videos. The project involves image preprocessing, training models on single and multiple vehicle images, and utilizing the YOLO model for real-time video detection.

## 🛠️ Tech Stack

- **AI/ML Frameworks:** TensorFlow, Keras, and PyTorch 
- **Model:** YOLO (You Only Look Once) for object detection  
- **Languages:** Python  
- **Tools:** OpenCV for image & video processing

---

## 🚀 Features

- Image preprocessing for training datasets  
- Trained on single and multiple vehicle images  
- Vehicle type detection and classification  
- Real-time detection in videos using YOLO model  
- Personal project exploring deep learning applications in computer vision

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/hanijawad04/Vehicle-Type-Detection.git
````

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Prepare dataset

* Add your vehicle images dataset in the specified folder
* Preprocess images using the provided scripts

### 4. Train the model

```bash
python train.py
```

### 5. Run detection on images/videos

```bash
python detect.py --source path/to/image_or_video
```
